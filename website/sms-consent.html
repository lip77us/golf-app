SMS Consent & Opt-In

Halved — operated by Paul Lipkin
Last updated: June 4, 2026

This page documents how users of the Halved mobile app consent to receive SMS text messages. Halved sends exactly one type of text: a one-time passcode (OTP) used to verify a phone number and sign the user in. No marketing or promotional messages are ever sent.

How users opt in

A user opts in by entering their own mobile number on the sign-in screen and tapping “Send code” to request a login passcode. The screen displays a clear consent disclosure before the user requests a code. Phone sign-in is optional — users may instead sign in with a username and password.

Message details

  • Use case: account verification / two-factor login (one-time passcodes).
  • Sample message: Your Halved verification code is: 123456
  • Frequency: only when the user requests a code — typically a few times per year (sign-in or new-device setup).
  • Rates: message and data rates may apply.
  • Opt-out: users can reply STOP, or simply stop requesting codes (and may use username/password sign-in instead).
  • Help: reply HELP or email support@halved.golf for assistance.
  • Privacy: mobile numbers are never sold or shared with third parties for marketing. See our Privacy Policy.
